🧠 이거 왜써요?

  1. 🗂️ 도메인(카테고리)별로 모아서 관리 가능
  1. 🔖 표 or 카테고리 형식 둘 다 지원
  1. 👓 상세 페이지에 request, params, response, example, status까지 기록 가능
  1. 💫 자유롭게 커스텀 가능
  1. ⏱️ “+ 추가하기” 한 번이면 새로운 API도 바로 등록 가능!
